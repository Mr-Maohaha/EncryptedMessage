In dieser Aufgabe seid ihr Geheimagenten (uuh) die eine Nachricht entschlüsseln müssen.
1) Die Daten werden aber nicht einfach so veröffentlicht. Nur wenn eure Node mit dem Namen
"secret_node" läuft, fängt die publisher node an ihre daten zu veröffentlichen.
Sie verschickt eine "encryptedMessage" aus diesem Package, indem ein char-array und ein key enthalten sind.
Jeder char hat auch einen Integer wert, dementsprechend wurde der key zu den einzelnen werten dazu addiert.

2) Um der Node mitzuteilen, dass die Nachricht entschlüsselt wurde, sollt ihr einen service client schreiben.
Der client soll den service "/data_check" anfragen. Der service der dabei verwendet wird ist ebenfalls aus
diesem package und heißt dataCheck. In der Variable "decryptedMessage" soll logischerweise der entschlüsselte
Text stehen. In "answer" steht dann, ob das Resultat richtig ist oder nicht. Die "answer" kann einfach geprinted
werden.


###English###
For this task you are a special agent (oooh) and you have to decrypt a message.
1) But, the data won't be available for everyone. The publisher node just start publishing if he recoginzes a
node with the name "secret_node". It will send an "encryptedMessage" from this package, which contains a char
array and a key. Every character can also be identified by an Integer, the key was added to that value to change
the outcome of the message.

2) To check your result you can call a service from the publisher node with the name "/data_check". The service
that has to be used is "data_check" from the prep_pkg. In has a string variable for the decrypted message text.
It also responds with a string to tell you, if the decryption is correct. This variable called "answer" should be 
printed.
